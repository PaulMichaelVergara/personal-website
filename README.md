# Personal Website
Welcome to my personal website! This is a modern minimalist theme with a dark and light mode toggle feature. The tech stack used includes React, Node.js, JavaScript, CSS, and HTML.

# Features
- Dark/Light Mode Toggle: Switch between dark and light mode with a single click.
- Contact Form: Use the contact form to get in touch with me. The form is powered by Email.js.
- About Me Section: Learn more about me in the about section. 
- Projects Section: Explore my projects and capabilities in the projects section.
